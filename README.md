# Kidney Stone Prediction using Machine Learning
## Description
This project predicts whether a person has kidney stone or not using a machine learning model.
The model is trained using a medical dataset and deployed using Flask web application.
## Features
- Data preprocessing
- Model training
- Prediction using trained model
- Flask web interface
- HTML frontend
## Tech Stack
- Python
- Flask
- Scikit-learn
- HTML / CSS
- NumPy / Pandas
## Dataset
Dataset taken from Kaggle:
https://www.kaggle.com/datasets/imtkaggleteam/kidney-stone-classification-and-object-detection

Dataset is not included in repo due to size limit.
## Accuracy
Model Accuracy: 92%   ← change if needed
